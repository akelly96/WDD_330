Week 8 Notes and Questions

Notes

  • Transform - lets you translate, rotate, scale, and/0r skew an element on the page.
  • translate - moves object up/down and left/right.
  • You need to include -webkit-transform and -ms-transform along with transform in order to accomodate older browsers.
  • scale changes the size of an object.
  • Rotates rotates the object by a certain amount of degrees.
  • Transition does not work in IE9.
  • A pre state is needed for transitions.
  • Transition duration is how long it will take to transition.
  • Transition timing functions controls the pace.
  • Transition delay delays the transition start.
  • @keyframe is used to define an animation
  • Canvas allows us to draw anysthing so we don't have to import image.
  • Canvas first should be added to the page using >canvas<
  • Drag and drop is an event listener
  • Dragstart is used to drag on item.
  • Drop is used to drop an item.
