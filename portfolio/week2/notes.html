Week 2 Notes and Questions

Notes

    Chapter 2
  • Var is scoped to the immediate function body while let is scoped to the immediate enclosing block.
  • camel case ariables with multiple words or use an underscore.
  • Setting an object variable to another variable is done by reference. Thus, changing one changes the other.
  • A backslash escapes special characters.
  • Infinity is any number bigger or smaller than +/- 1.8x10^308.
    Chapter 3
  • Always initilize an array.
  • The delete operator removes a specified object/value from an array.
  • Shift removes value/object from beggining of array. Unshift adds value/object to beginning of array.
  • Sets cannot include duplicates!
  • Map is like a dictionary in C++.
    Chapter 4
  • You need parentheses to invoke a function.
  • JavaScript has a forEach loop. Declare it using forEach(array, index) => {}
