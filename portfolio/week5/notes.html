Week 5 Notes and Questions

Notes

  • Exceptions are thrown when there is an error that causes the program to crash.
  • A warning occurs when there is an error that does not cause the program to crash.
  • Write JavaScript in strict mode to catch more errors.
  • There are many different ways to debug. Alerts, console.log(), debugger to name a few.
  • Remove references to debugger before turning in the final product.
  • JavaScript has try, catch, and finally statements
  • Try and catch statements work the same as other languages.
  • Finally will always be executed regardless of if an exception is thrown.
  • Use Jest to write unit tests.
