Week 9 Notes and Questions

Notes

Chapter 9
  • The Browser object model is a collection of properties/methods that contain info about the borwser/computer screen.
  • Window is a global object in the browser.
  • Window.location contains info about the URL.
  • window.reload reloads the page
  • window.assign loads another page
  • window.history allows for access to previous web pages
  • history.forward and . back help to navigate between pages
  • window.open will open a new browser.
  • document is contained within the window object
  • cookies allow for previous entrances into a URL to be remembered
  • document.cookie is used to store all cookies by a key and value
  • using a domain inside a cookie declaration makes it so only that domain can read that cookie
  • remove cookies by expiring them from a time in the past.
Chapter 14
  • data- is a way of embedding data in a web page using custom attributes ignored by the browser.
  • anything following the dash is up to the developer
  • .getattribute is used to retrieve the attribute associated w/ an object
  • navigator.geolocation gets the users location in the world
  • geolocation needs user permission to use
  • workers are used to do tasks in the background
  • If you have a large task to do, but click on multiple things that call multiple different functions, without workers, the program will freeze.
  • audio and video tags allow for videos or audio within the web browser
  • canvas allows the user to draw on the screen.
